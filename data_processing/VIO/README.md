## VIO

The code is used for getting actions from Visual Inertial videos (VIO). 

### Get Camera ID:
<pre>
python get_device.py
</pre>
### Record Trajectories:
Click mouse to start and click mouse to stop
<pre>
python record_trajectories.py
</pre>
### Visualization:
<pre>
python visualize_trajectories.py
</pre>
